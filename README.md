# Nipun & Hashintha Wedding Invitation

## Overview
A luxury interactive wedding invitation website for Nipun & Hashintha's wedding on November 20, 2026, at Hillview Resort, Sri Lanka.

## Features

### Premium Opening Experience
- 7-scene luxury intro sequence with particle effects
- Staggered text reveals for names
- Gold dot connection animations
- Monogram reveal with smooth transitions

### Interactive Elements
- **Hero Section**: Countdown timer, parallax background, floating particles
- **Gallery**: 5 high-quality images with lightbox experience
- **RSVP Form**: WhatsApp and Email integration with confetti celebration
- **Love Story**: Timeline of Nipun & Hashintha's journey
- **Wedding Program**: Detailed schedule of events
- **Venue Information**: Hillview Resort details
- **Dress Code**: Earth tone recommendations
- **Contact**: Direct contact for the couple

### Technical Excellence
- Modern CSS architecture with variables and components
- ES6+ JavaScript with module system
- GPU-accelerated animations
- Responsive design for all devices (320px - 1440px)
- Performance optimized with efficient particle systems

## Technology Stack

### CSS
- `variables.css` - Luxury color palette and design tokens

### JavaScript
- `main.js` - Application entry point and module coordination
- `intro.js` - Premium opening sequence

- `particles.js` - Hero section particle effects
- `gallery.js` - Interactive gallery with lightbox
- `rsvp.js` - RSVP form with WhatsApp integration

### Assets

- `assets/music/` - Background music (if needed)
- `assets/videos/` - Video content (if needed)

## Development Setup

### Prerequisites
- Modern web browser (Chrome, Safari, Firefox, Edge)
- Node.js (for validation script)

1. All CSS files are already compiled and ready
2. JavaScript modules are properly structured

```bash

```
